    # 前端常用库

    # css

    animate.css (opens new window) 一个简洁好用的css动画库

    magic.css (opens new window) 一个酷炫的css动画库

    iconfont (opens new window) 阿里图标(常用的icon都有)

    # js

    html2canvas (opens new window) 一个用于html转canvas的库

    echarts (opens new window) 一个基于 JavaScript 的开源可视化图表库

    javascript (opens new window) 用更合理的方式写 JavaScript的库(书写规范)

    pubsub-js (opens new window) 一个适用于react、vue的组件通信的库（发布订阅）

    js-cookie (opens new window) 一个简单轻量的操作cookie的js库

    nanoid (opens new window) 一个轻量的生成随机值得js库

    # vue

    vuesax (opens new window) 一个适用于 Vue 的 ui组件库

    vuepress (opens new window) Vue 驱动的静态网站生成器

    NProgress (opens new window) 一个体积超小的进度条插件

    vue l18n (opens new window) Vue.js 的国际化插件

    mitt (opens new window) Vue.js 的组件通信库（发布订阅）

    v-distpicker (opens new window) 一个简单易用的地区选择器

    vue-draggable (opens new window) 一个适用于 Vue 的拖拽DOM库

    vue-quill-editor (opens new window) 基于 Quill、适用于 Vue 的富文本编辑器，支持服务端渲染和单页应用。

    # react

    react-copy-to-clipboard (opens new window) 一个基于react封装的复制内容到剪切板的库

    react-highlight-words (opens new window) 一个基于react封装的高亮关键字的库

    Redux (opens new window) JavaScript 状态容器，提供可预测化的状态管理

    Redux Thunk (opens new window) Redux的异步处理中间件

    MobX (opens new window) 通过函数响应式编程使得状态管理变得简单和可扩展

    Redux Saga (opens new window) Redux中间件,用于管理应用程序 Side Effect(副作用，例如异步获取数据，访问浏览器缓存等)

    Dva (opens new window) 一个基于 redux 和 redux-saga 的数据流方案

    G2Plot (opens new window) G2Plot 是开箱即用、易于配置、具有良好视觉和交互体验的通用统计图表库。

    # UI

    iview (opens new window) 一套基于 Vue.js 的高质量UI组件库

    element (opens new window) Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

    antd (opens new window) 企业级产品设计体系，创造高效愉悦的工作体验

    antd-vue (opens new window) 这里是 Ant Design 的 Vue 实现，开发和服务于企业级后台产品。

    swiper (opens new window) 开源、免费、强大的触摸滑动插件

    bootstap (opens new window) 利用 Bootstrap 构建快速、响应式的网站

    vant (opens new window) 轻量、可靠的移动端 Vue 组件库

    layui (opens new window) 由职业前端倾情打造，面向全层次的前后端开发者，易上手开箱即用的 Web UI 组件库

    museui (opens new window) 基于 Vue 2.0 优雅的 Material Design UI 组件库

    mintui (opens new window) 基于Vue.js的移动端组件库
